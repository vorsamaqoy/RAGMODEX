# RAGMODEX Frontend

React/Vite client for the RAGMODEX molecular ML workbench.

## Development

```bash
npm install
npm run dev
```

The development server runs on `http://localhost:5173` and proxies `/api` requests to the FastAPI backend on `http://localhost:8000`.

## Build

```bash
npm run build
```

See the root README for full project setup and backend instructions.
